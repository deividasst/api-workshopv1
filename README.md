## Requirements of API automation workshop
1. Install node.js and npm: https://nodejs.org/en/download/prebuilt-installer/current
2. Check by running :
  node -v
  npm -v
3. Install yarn:
  npm install --global yarn
4. Check by running: 
   yarn -v
5. Git clone repo
6. yarn install //install dependecies
